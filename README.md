# 💰 Student Expense Tracker

A full-stack Expense Tracker application built using HTML, CSS, JavaScript, Node.js and Express.js.

## 🚀 Features

- Add new expenses
- Delete expenses
- Categorize expenses
- Track total spending
- Store data using JSON file
- Simple and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## 📂 Project Structure

```
student-expense-tracker
│
├── public
│   ├── index.html
│   ├── style.css
│   └── script.js
│
├── server.js
├── data.json
├── package.json
```

## ▶️ Run Locally

```bash
npm install
node server.js
```

Open:

```
http://localhost:3000
```
